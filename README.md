# 🎬 IMDB Movie Review Sentiment Analysis (RNN + Streamlit)

An end-to-end **Deep Learning project** that predicts whether a movie review is **Positive 😊 or Negative 😠** using a **Simple RNN model**, with an interactive **Streamlit web application** for real-time predictions.

## 📌 Project Overview

Understanding user sentiment is crucial in today’s data-driven world.
This project uses **Natural Language Processing (NLP)** and **Deep Learning** to analyze IMDB movie reviews and classify them into positive or negative sentiments.

The model is trained using a **Simple Recurrent Neural Network (RNN)**, which is effective for sequential text data.

---

## 🧠 Model Details

* Model Type: **Simple RNN**
* Dataset: **IMDB Movie Reviews Dataset**
* Task: **Binary Sentiment Classification**
* Framework: **TensorFlow / Keras**
* Text Processing: Tokenization + Padding
